# `alphaclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
alphaclaw pairing list whatsapp
alphaclaw pairing approve whatsapp <code> --notify
```
