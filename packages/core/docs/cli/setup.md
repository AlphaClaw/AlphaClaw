# `alphaclaw setup`

Initialize `~/.alphaclaw/alphaclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
alphaclaw setup
alphaclaw setup --workspace ~/.alphaclaw/workspace
```

To run the wizard via setup:

```bash
alphaclaw setup --wizard
```
